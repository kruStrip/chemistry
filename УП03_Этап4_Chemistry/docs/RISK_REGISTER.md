| Проект недоступен после перезапуска | После `restart.bat` сервисы должны подняться с `restart: unless-stopped` | Средняя | Высокое | Проверять `check_deploy.bat`, health, логи; политика restart в compose |
| Ошибки API приводят к 500 | Неверный JSON на login давал 400 (не 500); несуществующий курс — 404 | Низкая | Высокое | Сохранять ProblemDetails, добавить интеграционные тесты в CI |
| Медленная загрузка страницы | Lighthouse Performance; GET /api/v1/reviews ~2.7s в логах | Средняя | Среднее | Оптимизировать запросы EF (QuerySplitting), кэш Redis |
