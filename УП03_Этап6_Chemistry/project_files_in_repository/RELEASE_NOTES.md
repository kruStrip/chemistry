# RELEASE_NOTES.md — Chemistry v0.1.1

**Дата:** 10.06.2026  
**Версия:** 0.1.1  
**Этап:** УП.03 — CI/CD, релиз и инцидент поддержки (этап 6)

## Что исправлено

- **Вход в личный кабинет:** при неверном email/пароле пользователь видит понятное сообщение «Неверный email или пароль», а не общую фразу про истёкшую сессию.
- **API auth/login:** в ответе 401 добавлен `detail` на русском языке.
- **CI/CD:** скрипты релиза, `release-check`, единый workflow `ci.yml`.

## Что проверить после обновления

1. Обновить образы или пересобрать demo: `scripts\deploy.bat`.
2. Открыть http://localhost:3000/login и ввести заведомо неверный пароль — должна появиться ошибка «Неверный email или пароль».
3. Запустить `scripts\release-check.bat` — все проверки должны пройти.
4. Убедиться, что `GET http://localhost:5000/health` возвращает `"status":"Healthy"`.

## Как обновить

```bat
cd chemistry-backend
git pull
scripts\release-check.bat
scripts\deploy.bat
```

Для production — через GitHub Actions и тег `v0.1.1`.

## Известные ограничения

- Demo-стенд использует mock-интеграции (платежи, почта).
- На пустой БД каталог курсов пуст до ручного наполнения.
- npm audit: остаются moderate/high в транзитивных зависимостях (см. этап 5).
