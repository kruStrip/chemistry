# УП.03 — Этап 3: Chemistry Na Easy

Развертывание demo-стенда через Docker (вариант C).

## Быстрый запуск

```bat
cd chemistry-backend
scripts\deploy.bat
```

Открыть: http://localhost:3000

## Содержимое папки

| Путь | Описание |
|------|----------|
| `repo_link.txt` | Ссылки на GitHub |
| `public_url.txt` | Адреса demo-стенда |
| `docs/` | DEPLOYMENT.md, DEMO_GUIDE.md, RELEASE_NOTES.md, отчёт |
| `screenshots/` | 9 обязательных скриншотов |
| `scripts/` | deploy / restart / check_deploy / build_release |
| `release/` | project_release.zip, demo_readme.txt |
| `project_files_in_repository/` | Копии файлов для репозитория |
